# dotfiles

My dotfiles inspired by jared palmer's dotfiles

## Installation

```sh
cd ~
git clone https://github.com/alexthomsonnz/dotfiles.git && cd dotfiles && ./sync.sh
```

## Brew Stuff

```bash
# Essentials
brew install git
# DB
brew install postgresql
brew install pgcli
brew install redis
# Fish/CLI
brew install z
brew install fish
# Ops
brew install awscli
brew install heroku
```
